Class ActiveStorage::Attachment < ActiveStorage::Record

  • activestorage/app/models/active_storage/attachment.rb

Attachments associate records with blobs. Usually that's a one record-many blobs relationship, but it is possible to associate many different records with the same blob. A foreign-key constraint on the attachments table prevents blobs from being purged if theyâ€™re still attached to any records.

Attachments also have access to all methods from ActiveStorage::Blob.

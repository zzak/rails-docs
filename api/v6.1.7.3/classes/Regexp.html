multiline?() Link

Returns true if the regexp has the multiline flag set.

(/./).multiline?  # => false
(/./m).multiline? # => true

Regexp.new(".").multiline?                    # => false
Regexp.new(".", Regexp::MULTILINE).multiline? # => true
